Project Android

Aplikasi Kuningan Asri

Aplikasi ini merupakan aplikasi yang pertama kali saya buat, menggunakan Android Studio, berisi tentang tempat-tempat wisata yang ada di Kuningan-Jawa Barat. Aplikasi ini dibuat sebagai syarat menyelesaikan pelatihan Online Dicoding.
Beberapa hal yang saya pelajari disini ialah :

  1. Pengenalan Android Studio
  2. Penggunaan Activity, mulai dari menuliskan logika dasarnya, sampai penggunaan saveInstanceState.
  3. Penggunaan Intent serta bagaimana cara mengirim data dengan menggunakan Parcelable.
  4. Layouting menggunakan LinearLayout, FrameLayout, RelativeLayout, dan TableLayout.
  5. Styling and Theming, salah satunya penggunaan ripple effect
  6. Penggunaan RecyclerView dengan berbagai tipe layout, seperti tipe List, Grid, maupun CardView.

Aplikasi Katalog Film

Sesusai namanya, aplikasi ini berisi katalog film, dimana datanya saya ambil dari themoviedb.org. Aplikasi ini dibuat sebagai syarat seleksi dari program IndosatxDicoding.
Beberapa hal yang saya pelajari disini ialah :

  1. Prinsip Dasar Android seperti Activity, Fragment, macam-macam Intent, dan ListView.
  2. UI dan UX Aplikasi. Penggunaan layout, style, theme, localization, CustomView, RecyclerView dan Material Design seperti ActionBar, NavigationDrawer, BottomNavigation, dan TabLayout.
  3. Testing, diantaranya Unit Testing dan Instrumentation Testing.
  4. Background Process, seperti Service, Broadcast Receiver, AlarmManager, JobScheduler, dan WorkManager, serta mengambil data dari restful API yang berformat JSON.
  5. Local Data Storage, untuk menyimpan data di Android seperti File, SharedPreference, database SQLite.
